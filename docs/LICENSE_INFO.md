# Informações sobre Licenciamento - Excel-ai

## Visão Geral

O Excel-ai é um software proprietário com sistema de licenciamento integrado.

## Modelo de Licenciamento

### Licença de Uso

O Excel-ai utiliza um sistema de validação de licença para controlar o uso da aplicação.

**Características**:
- Verificação de licença no startup
- Validação periódica durante o uso
- Sistema de licença baseado em chave

### Obtenção de Licença

Para obter uma licença válida:

1. **Contato**: Entre em contato com o autor
   - Email: jefferson@hiposystem.com.br
   
2. **Avaliação**: Discuta suas necessidades e casos de uso

3. **Licenciamento**: Receba sua chave de licença

4. **Ativação**: Configure a licença na aplicação

## Tipos de Licença

### Licença de Desenvolvimento

**Uso Permitido**:
- Desenvolvimento e teste
- Ambiente de desenvolvimento pessoal
- Avaliação da aplicação

**Limitações**:
- Não pode ser usado em produção
- Uso individual apenas
- Período limitado de avaliação

### Licença Comercial

**Uso Permitido**:
- Uso em produção
- Uso comercial
- Múltiplos usuários (conforme plano)

**Modalidades**:
- **Single User**: Uma estação de trabalho
- **Team**: Até 5 usuários
- **Enterprise**: Número ilimitado de usuários

## Privacidade e Dados

### Coleta de Dados

O Excel-ai:

**NÃO coleta**:
- Dados pessoais
- Conteúdo de planilhas
- Histórico de conversas
- Informações sensíveis

**Armazena Localmente**:
- Configurações da aplicação
- Histórico de conversas (opcional)
- Chaves de API (localmente)

### Dados Enviados a Terceiros

**APIs de IA**:
- Mensagens do usuário
- Dados de planilhas (quando necessário para análise)
- Contexto da conversa

**Nota**: Dados são enviados apenas para provedores de IA configurados (OpenRouter, Groq, Google, etc.) e estão sujeitos às políticas de privacidade desses provedores.

## Conformidade

### LGPD (Lei Geral de Proteção de Dados)

O Excel-ai foi desenvolvido com respeito à LGPD:

- Minimização de dados coletados
- Armazenamento local (não em nuvem)
- Controle do usuário sobre seus dados
- Transparência sobre uso de dados

### GDPR

Para usuários na União Europeia:

- Direito ao esquecimento (deletar conversas)
- Direito de acesso (visualizar dados salvos)
- Direito à portabilidade (exportar conversas)
- Processamento legal de dados

## FAQs sobre Licenciamento

### Posso usar em múltiplas máquinas?

Depende da licença:
- Single User: Até 2 máquinas do mesmo usuário
- Team/Enterprise: Conforme número de licenças

### Preciso de internet para usar?

- Internet necessária para:
  - Validação inicial de licença
  - Uso de APIs de IA
  - Atualizações
- Uso offline limitado (após validação)

### O que acontece se minha licença expirar?

- Notificação 30 dias antes
- Período de graça de 7 dias
- Após expiração: Funcionalidade limitada
- Dados salvos não são perdidos

### Posso transferir minha licença?

Sim, mediante:
- Solicitação formal
- Aprovação do autor
- Possível taxa de transferência

### Há desconto para educação/ONGs?

Sim! Entre em contato para:
- Desconto educacional
- Licenças para ONGs
- Programas especiais
